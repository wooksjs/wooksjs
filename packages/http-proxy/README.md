Composable HTTP proxy for [@wooksjs/event-http](https://github.com/wooksjs/wooksjs/tree/main/packages/event-http). Proxy requests with fine-grained control over headers and cookies filtering.

## Installation

```sh
npm install @wooksjs/http-proxy
```
